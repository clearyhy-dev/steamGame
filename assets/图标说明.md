# 应用图标说明

已为 Steam Deal Alert 设计并生成图标：**app_icon_512.png**（512×512，适合作为应用图标源图）。

## 替换为启动器图标

若要将该图设为手机桌面/启动器图标，需生成各尺寸并替换 Android 资源：

1. **用 Android Studio**  
   - 用 Android Studio 打开项目下的 `android` 目录  
   - 右键 `app` → **New** → **Image Asset**  
   - 选择 **Launcher Icons**，Path 指向本目录的 **app_icon_512.png**  
   - 生成后会自动更新 `res/mipmap-*` 下的 `ic_launcher.png`

2. **用在线工具**  
   - 使用 [App Icon Generator](https://appicon.co/) 等网站上传 **app_icon_512.png**  
   - 下载 Android 包并解压，将各 `mipmap-*` 中的 `ic_launcher.png` 覆盖到  
     `android/app/src/main/res/` 下对应目录

替换完成后重新打包即可看到新图标。
