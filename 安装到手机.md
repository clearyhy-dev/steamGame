# 把 APK 安装到手机

## 使用 D 盘环境打包并安装（推荐）

已配置使用 **D:\development\flutter**、**D:\dev-config\gradle**、**D:\dev-config\pub**。

**一键打包并安装到已连接手机：**

```powershell
cd e:\steamGame
.\build-and-install-with-d.ps1
```

或手动设置环境后打包：

```powershell
$env:PATH = "D:\development\flutter\bin;" + $env:PATH
$env:GRADLE_USER_HOME = "D:\dev-config\gradle"
$env:PUB_CACHE = "D:\dev-config\pub"
cd e:\steamGame
flutter build apk --release
```

打包完成后，APK 位置：

**`e:\steamGame\build\app\outputs\flutter-apk\app-release.apk`**

---

## 二、安装方式

### 方式 1：数据线 + 电脑安装（推荐）

1. 手机用 USB 连接电脑，并开启「USB 调试」：
   - 设置 → 开发者选项 → 打开「USB 调试」
2. 在电脑上执行（在项目目录或任意目录均可）：

```powershell
adb install -r "e:\steamGame\build\app\outputs\flutter-apk\app-release.apk"
```

`-r` 表示若已安装则覆盖安装。

若未安装 adb：可从 [Android SDK Platform-Tools](https://developer.android.com/studio/releases/platform-tools) 下载，把 `platform-tools` 加入系统 PATH，或使用完整路径运行 `adb`。

---

### 方式 2：把 APK 拷到手机再安装

1. 把 **`app-release.apk`** 复制到手机（微信/QQ 发给自己、U 盘、云盘等）。
2. 在手机上用文件管理器找到该 APK，点击安装。
3. 若提示「不允许安装未知应用」，在设置里对该来源（如「文件管理」）允许安装未知应用后再试。

---

### 方式 3：用 Flutter 直接装到已连接设备

手机连上电脑并开启 USB 调试后，在项目目录执行：

```powershell
flutter install --release
```

会先构建再安装到当前连接的设备（无需先手动找 APK 路径）。
