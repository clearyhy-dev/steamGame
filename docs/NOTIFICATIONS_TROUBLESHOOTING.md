# 手机收不到提醒的常见原因与处理

应用会按**你所选地区/语言对应的时区**，在**每天 11:00、17:00、20:00** 各发送一次「今日折扣」类提醒；Pro 用户还会在同时段收到「愿望单降价」提醒。通知文案会随应用内选择的语言显示。

## 1. 通知权限未开启

- 打开 **系统设置 → 应用 → Steam AI Deal Alert → 通知**，确认通知已开启。
- 若首次安装时点了「不允许」，需要在这里手动打开。

## 2. 至少打开过一次应用

- 每日任务是在**应用启动时**注册的；若安装后从未打开过应用，当天不会收到提醒。
- 建议安装后**至少打开一次**并允许通知，到下一个 11:00 / 17:00 / 20:00 应能收到。

## 3. 时区与提醒时刻

- 提醒时间按**应用内选择的地区/语言对应时区**的 **11:00、17:00、20:00**。
- 若你在应用里选了「美国」等，这三个时刻会是该地区时间；可在 **个人页 → 地区** 中调整。

## 4. 代码与渠道说明

- **每日折扣**使用通知渠道 `daily_channel`，首次启动会创建，到下一个 11:00 / 17:00 / 20:00 应能正常收到「今日折扣」提醒。
- 所有通知文案均随应用语言切换（中/英等），样式统一。

若以上都确认无误仍收不到，请查看系统「通知历史」里是否有被静默拦截，或尝试卸载后重新安装并重新授权通知。
